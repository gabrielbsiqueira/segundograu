Definição


Chama-se função quadrática, ou função polinomial do 2º grau, qualquer função f de IR em IR dada por uma lei da forma f(x) = ax2 + bx + c, onde a, b e c são números reais.

Exemplos:
  • f(x) = 3x² - 4x + 1, onde a = 3, b = - 4 e c = 1.

  • f(x) = x² -1, onde a = 1, b = 0 e c = -1.

  • f(x) = 2x² + 3x + 5, onde a = 2, b = 3 e c = 5.

  • f(x) = - x2 + 8x, onde a = -1, b = 8 e c = 0.

  • f(x) = -4x², onde a = - 4, b = 0 e c = 0.

O gráfico de uma função polinomial do 2º grau, y = ax² + bx + c, com a diferente de 0, é uma curva chamada parábola.
Nesse caso, é notavel que y = 5 (valor na linha vertical) e x = 2 (valor na linha horizontal).
