Coeficiente "a"


  • Numa equação do 2°grau o "a" é o coeficiente de x², sendo este, obrigatóriamente diferente de 0, pois caso contrárico está se tornaria uma equação do 1°grau (bx + c = 0).

  • Em uma função, dentro do gráfico, esta determina a concavidade da parábola sendo que:

(a > 0) Concavidade para cima

(a < 0): Concavidade para baixo
